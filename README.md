# Prerequirements
1. Computer
2. GCC C compiler
3. Raylib library

# Building
For simped users: just press this button: @![GREEN]@![SHELL,cc -o main main.c -lraylib -lm,Build]@![NOC] \
For non-simped users:
```shell
$ cc -o main main.c -lraylib -lm
```

# Using
```shell
$ ./main
```
